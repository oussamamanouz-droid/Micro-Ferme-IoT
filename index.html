Micro-Ferme Agricole
Intelligente

Système autonome de gestion des cultures basé sur les technologies IoT, alimenté par énergie solaire et réseau électrique (secteur) en parallèle — conçu selon les principes de l'ingénierie système (Prof. M. Rafi).

🔌 IoT / ESP32 ☀️ Énergie Solaire 💧 Irrigation Auto 📱 Supervision Mobile 🌡️ Température 💧 Humidité Air 🌿 Type de Plante
🎯

Objectif Principal

Surveiller température et humidité de l'air, adapter l'irrigation selon le type de plante cultivée.

💧

Eau Optimisée

Irrigation automatique selon les données capteurs, réduisant le gaspillage.

☀️

Double Alimentation

Alimentation hybride : énergie solaire (panneau + batterie) + réseau secteur 220V en secours automatique.

📡

Supervision à Distance

Interface web/mobile pour le suivi et le contrôle à distance du système.

Contexte

Problématique Identifiée

Les petites exploitations agricoles font face à trois défis majeurs : une irrigation non optimisée entraînant du gaspillage d'eau, le manque de suivi en temps réel des conditions de culture, et la dépendance aux énergies classiques coûteuses et polluantes.

Démarche IS

Approche Ingénierie Système

Ce projet applique la méthode du Cycle en V : expression du besoin → analyse fonctionnelle (bête à cornes, pieuvre, FAST) → spécification des exigences → conception de l'architecture → validation du prototype.

Architecture Logicielle & Flux de Données
Organisation du firmware embarqué, couches logicielles et protocoles de communication
Extrait de Code Embarqué (Arduino IDE)
// ── Micro-Ferme Intelligente IoT · ESP32 · M.RAFI ──────────────────
#include <WiFi.h>
#include <PubSubClient.h>   // MQTT
#include <DHT.h>

// ── Constantes ──────────────────────────────────────────────────────
const int   PIN_DHT      = 4;     // GPIO4  → DHT22
const int   PIN_RELAIS   = 26;    // GPIO26 → Module relais
// Seuils par type de plante (configurable via Blynk)
const int   SEUIL_TOMATE = 60;  // % humidité air
const int   SEUIL_SALADE = 70;  // % humidité air
const int   SEUIL_CACTUS = 30;  // % humidité air
int    seuil_actif  = SEUIL_TOMATE; // Défaut
const long  INTERVAL_MS  = 5000;  // Lecture capteurs toutes les 5s

DHT    dht(PIN_DHT, DHT22);

void loop() {
  // 1. Acquisition
  float temperature  = dht.readTemperature();
  float hum_air      = dht.readHumidity();  // % humidité de l'air

  // 2. Décision irrigation selon type de plante
  if (hum_air < seuil_actif) {
    digitalWrite(PIN_RELAIS, LOW);   // Active LOW → pompe ON
    delay(10000);                     // Arrosage 10 secondes
    digitalWrite(PIN_RELAIS, HIGH);  // Pompe OFF
  }

  // 3. Envoi MQTT vers ThingSpeak
  String payload = "field1=" + String(temperature)
                 + "&field2=" + String(hum_air)
;
  mqtt.publish("channels/XXXX/publish", payload.c_str());

  delay(INTERVAL_MS);
}
